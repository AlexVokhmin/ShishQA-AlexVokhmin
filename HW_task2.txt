HTTP (протокол передачи гипертекста) – прикладной протокол передачи данных, основанный на архитектуре клиент-сервер, при котором от клиента происходит запрос к серверу, а от сервера приходит ответ клиенту.
HTTPS (защищенный протокол передачи гипертекста) – защищенный протокол передачи данных. Расширение протокола HTTP, выполняющее шифрование передаваемых данных посредством криптографического протокола SSL или TLS.

Версии HTTP:
HTTP 0.9 – Экспериментальная версия протокола обмена данными посредством протокола TCP, реализующего архитектуру клиент-сервер
HTTP 1.0 – Первая официальная версия протокола
HTTP 1.1 – Протокол, позволяющий отправлять сразу несколько запросов в течение одного соединения по TCP и требующий указывать имя хоста при отправке запроса на сервер
HTTP 2.0 – Бинарная версия протокола, поддерживающая расстановку приоритетов запросов и их мультиплексирование  
HTTP 3.0 – Применение в качестве транспортного протокола UDP вместо TCP.
